## 0.2.0

* **RTL Source Selection**: Added support for both local USB RTL-SDR (via rtl_tcp_andro) and remote RTL-TCP servers
* **Structured Call Events**: Replaced log parsing with direct polling of DSD-Neo state for accurate call detection
* **New RadioGroup UI**: Updated to Flutter 3.38 RadioGroup API for RTL source selection
* **Play Store Integration**: Added button to install rtl_tcp_andro driver from Play Store
* **Landscape Only**: Enforced landscape orientation for optimal scanner display

## 0.1.0

* Initial release of Pocket25 - P25 Phase 1 Scanner
* Scanner screen with real-time talkgroup activity
* Log screen with color-coded DSD output
* Settings screen for RTL-TCP configuration
* Automatic log parsing for talkgroup detection
