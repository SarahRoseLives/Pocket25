2. Enter RTL-TCP details

5. Switch to **Scanner** tab â†’ See "Listening for activity..."

### Stopping Scanner
1. Go to **Settings** tab
2. Tap **Stop Scanner**
3. Scanner tab shows "Start scanner to see activity"

### Viewing Logs
1. Start scanner (as above)
2. Switch to **Log** tab
3. Colored text scrolls as DSD outputs
4. Tap â¬‡ï¸ to jump to bottom

### Copy Log Line
1. Go to **Log** tab
2. Long-press any line
3. Text becomes selectable
4. Copy with system controls
